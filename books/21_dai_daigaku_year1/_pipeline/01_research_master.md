### Year 4 (Sách 24)

## 4. Hizashi system (sách 24 climax)

### Architecture
- **Mobile**: Flutter (iOS + Android)
- **Backend**: FastAPI (Python) + Pydantic + SQLAlchemy
- **DB**: PostgreSQL + pgvector for embeddings
- **AI**: Claude API (claude-3-7-sonnet) for tutor + embedding cho semantic search
- **Storage**: Cloudflare R2 cho audio/image
- **Auth**: Supabase Auth hoặc custom JWT
- **Deploy**: Docker + AWS ECS hoặc Vercel + Render

### Features
- **Study Course system**: Khoá học nhiều cấp (N5→N1), lý thuyết markdown + dialogue
- **Question Set**: 20+ question types (chon_dap_an, kanji_yomi, reading_passage, listening, ...)
- **SRS**: Spaced repetition cho vocab/kanji
- **AI Tutor**: Claude Q&A, giải thích ngữ pháp Việt-Nhật
- **Reading Passage**: Bài đọc N3-N1 với glossary inline + comprehension questions
- **Mogishiken**: Đề thi thử JLPT đầy đủ
- **Voice**: TTS Nhật + recording feedback
