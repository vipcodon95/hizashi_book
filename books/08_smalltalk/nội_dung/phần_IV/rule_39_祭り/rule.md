# Rule 39 — Lễ hội VN / ベトナムの祭り

> **Luận điểm.** JP yêu 祭り của họ — và rất tò mò "VN có tương đương gì?". 4 lễ hội kể được sâu = **Tết Trung Thu (中秋節 — gần Obon JP), Lễ Hùng Vương (建国記念日相当), Quốc Khánh 2/9, Tết Đoan Ngọ (端午の節句相当 nhưng VN khác)**. Tránh political triggers quanh 2/9 và 30/4.

---

## 🧠 Tâm lý JP (4 dòng)

- 祭り JP = một trong 3 topic JP cảm xúc nhất (cùng 食 + 故郷). Bridge với VN festivals = mở conversation gold.
- Trung thu VN = "Tết của trẻ con" gần lăng kính tâm tình của Obon JP (gia đình + ánh trăng) → bridge tự nhiên.
- 2/9 (Quốc Khánh) là **dây nhạy** — chỉ dùng nếu khách hỏi, focus vào **nghỉ lễ + bắn pháo hoa**, KHÔNG sa đà chính trị.
- Senior JP rất thích nghe "tradition that survived" — múa rồng, đèn ông sao, bánh trung thu.

## 💬 4 Scenarios — dialogue thực chiến

### 📍 Scenario 1 — Trung thu (Tết Thiếu Nhi) — bridge với Obon

| Speaker | Câu |
|---------|-----|
| **松本** | 「ズンさん、9月7日は祝日?」 |
| **ズン** | 「祝日ではないですが、**中秋節(Tết Trung Thu)**で、**ベトナムの子供のお祭り**です。」 |
| **松本** | 「あ、中秋節ってアジア共通?」 |
| **ズン** | 「中国・韓国・ベトナム・台湾で祝います。但し、**ベトナムでは"子供メイン"**で、**親が子供のために提灯と月餅を買う日**です。」 |
| **松本** | 「日本のお盆と何が違うの?」 |
| **ズン** | 「お盆は**祖先供養が中心**ですよね。**ベトナムの中秋は"次世代"が中心**で、雰囲気が180度違います。**夕方、子供たちが赤い星型の提灯**(Đèn ông sao)を持って**獅子舞・龍舞**(Múa Lân)を見ながら街を歩きます。」 |
| **松本** | 「ええ、可愛い!月餅は日本にもあるけど、ベトナムのは違うの?」 |
| **ズン** | 「**バインチュンチュー(Bánh Trung Thu)**、表面にお花の模様で、中身は**蓮の実、カラスミ卵黄、緑豆あん、カスタード**等。**カラスミ入り**は日本人結構衝撃受けます(笑)。」 |
| **松本** | 「カラスミ?それは食べてみたい!」 |

> **VN:** Bridge "Trung thu = Tết trẻ con / Obon = lễ tổ tiên" → khách Nhật instant phân biệt + thấy fascinating.

### 📍 Scenario 2 — Lễ Hùng Vương (Quốc giỗ — 10/3 âm)

| Speaker | Câu |
|---------|-----|
| **加藤** | 「ベトナムの建国記念日みたいなのって、いつ?」 |
| **ズン** | 「**フンヴオン王の命日(Giỗ Tổ Hùng Vương)**、旧暦3月10日です。**4000年前の伝説の初代王様の命日**で、毎年祝日になります。」 |
| **加藤** | 「4000年前!?それは深いね。」 |
| **ズン** | 「日本の天皇陛下と似た位置づけで、**国の祖**として崇敬されています。**フート省のフンテンプル(Đền Hùng)**で大祭が行われます。」 |
| **加藤** | 「岐阜の伊勢神宮みたいな感じ?」 |
| **ズン** | 「はい、**国家鎮護の聖地**って感じ近いと思います。**4月頃**で、私も子供の頃**学校で『コン・ロン・チャウ・ティエン(龍と仙女の子孫)』**っていう建国神話を習いました。**龍の父と仙女の母から生まれた100人の兄弟が南北に分かれた**っていう神話です。」 |
| **加藤** | 「100人兄弟!それは岐阜の山岳信仰系の伝説に近いね、面白い。」 |

📝 **Cách kể an toàn:** stick với 4000 năm + 神話 + 聖地 — KHÔNG sa đà vào "Bắc-Nam tách 100 con" theo lăng kính chính trị.

### 📍 Scenario 3 — Quốc Khánh 2/9 — kể an toàn

*tone trung tính, fact only · focus vào hiện tại, không quá khứ*

| Speaker | Câu |
|---------|-----|
| **佐藤** | 「9月2日は何の祝日?」 |
| **ズン** | 「**ベトナム独立記念日**で、**国民の祝日**です。**ハノイのバーディン広場で式典**、**夜は花火**があります。」 |
| **佐藤** | 「ふむ、戦後の話やね…ベトナムの方たち、どう過ごす?」 |
| **ズン** | 「**3-4日連休になることが多くて**、**家族で帰省 or 国内旅行**が一般的です。私の家族も**サパに行ったり、ハロン湾に行ったり**します。」 |
| **佐藤** | 「ああ、シルバーウィークみたいなもんやね。」 |
| **ズン** | 「まさにそんな雰囲気です。**夜は街のあちこちで赤い旗が出る**ので、**ハノイのオペラハウス周辺**は写真スポットとして人気です。」 |
| **佐藤** | 「ええなあ、出張のタイミングが合えば見てみたい。」 |
| **ズン** | 「但し**この時期はホテル予約が取りにくい**ので、出張なら**避けるか早めに**お願いします。」 |

📝 **NG:** "アメリカに勝った", "戦争の勝利", "南進" → tuyệt đối tránh. **OK:** "祝日 + 連休 + 花火 + 帰省" — life today.

### 📍 Scenario 4 — So sánh festival JP-VN (multi-festival)

| Speaker | Câu |
|---------|-----|
| **山本** | 「VNのお祭りって、関西の祭りに似たの何かあるん?」 |
| **ズン** | 「いくつか面白い対応関係あります。**①端午の節句(5/5) → VNの『テト・ドアンゴー(Tết Đoan Ngọ)』**は同じ日付ですが、**鯉のぼりじゃなくて"虫殺しの日"**です。」 |
| **山本** | 「虫殺し!?」 |
| **ズン** | 「(笑)**朝5時に酒漬けのもち米とフルーツを食べて、体内の寄生虫を殺す**っていう民間療法的な日です。私の祖母も毎年やってました。」 |
| **山本** | 「面白すぎる…他は?」 |
| **ズン** | 「**②七夕** — VNにもあって**七夕=Thất Tịch (7/7 旧暦)**、**牽牛織女の伝説**は同じ。**雨が降ると織姫が泣いている**って解釈、共通です。」 |
| **山本** | 「同じやん!」 |
| **ズン** | 「**③盆踊り**みたいなのは**VNにはない**けど、**フエの宮廷舞踊**や**チャムパ族の収穫舞**みたいなのが地域祭りで見られます。」 |
| **山本** | 「祭りの違いって、文化の違いがダイレクトに出るね。」 |

📝 **Insight:** đối ứng JP-VN festivals = topic vàng cho khách Kansai (mê 祭り). Té Đoan Ngọ "diệt sâu bọ" = bizarre, dễ nhớ.

## ⚠ NG — tuyệt đối tránh

- 2/9 → "戦勝記念日" / 戦争 / 米国敗退 → vùng cấm chính trị tuyệt đối.
- 30/4 (Reunification Day) cũng tránh nói sâu — focus 連休 only nếu có.

- "Bắc-Nam 100 con" theo lăng kính chính trị → giữ ở mức 神話.
- Mô tả múa rồng / Lân là "TQ traditions" → nhạy cảm, focus VN local development.
