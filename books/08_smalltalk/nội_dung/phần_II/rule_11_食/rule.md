# Rule 11 — Ăn — món địa phương / 食の話題

> **Luận điểm.** 食 = **topic gold tuyệt đối an toàn**. Người Nhật **cuồng** chia sẻ 名物 quê họ. Mỗi vùng có "ấn tượng món" riêng — nhớ được 3-5 món/tỉnh = bạn không bao giờ bí topic. Đây là phần dễ nhất + đem lại nhiều ấn tượng nhất trong sách 08.

---

## 🧠 Tâm lý JP (4 dòng)

- 名物 (đặc sản) = niềm tự hào vùng quê. Hỏi đúng món = trúng tim.
- JP thích **so sánh nội bộ** (大阪お好み焼 vs 広島お好み焼 — hai bên cãi nhau yêu) — bạn join vào = thân.
- Senior 50-60t đặc biệt nostalgic về 子供の頃の味 — món ăn tuổi thơ.
- Câu cấm: "ベトナム料理の方が美味しい" / "[món A] không hợp khẩu vị tôi" → ngưng.

---

## 🎬 Bối cảnh

Dũng + Hà CTO onsite Tokyo + Osaka 5 ngày. Dinner mỗi tối với khách quê khác nhau — Tokyo (Matsumoto), Osaka (Yamamoto), Hiroshima (Hiroshi), Nagoya (Ito), Gifu (Kato). Phải xoay topic 食 để mỗi người được kể về đặc sản của vùng họ.

## 💬 4 Scenarios — dialogue thực chiến

### 📍 Scenario 1 — お好み焼 戦争: Hiroshima vs Osaka

*izakaya Osaka, đã uống 2 ly · cười · thận trọng + ngoại giao · rạng rỡ · không chịu*

| Speaker | Câu |
|---------|-----|
| **山本** | 「ズンさん、お好み焼きは大阪派?広島派?」 |
| **広島** | 「これは戦争になるで(笑)」 |
| **ズン** | 「両方食べたことがあって、それぞれ違う美味しさですよね。**広島は層になってて**【1】、**大阪は混ぜて焼く**【2】、と聞きましたが…」 |
| **広島** | 「お、よく分かってる!広島は**そば入り**【3】が基本、キャベツも倍以上入る。お好み焼きとは別物よ。」 |
| **山本** | 「いやいや、大阪のは**ふんわり**してて、生地と具が一体になっとる。広島のはミルフィーユや(笑)」 |
| **ズン** | 「広島だと**みっちゃん総本店**、大阪は**ぼてぢゅう**って聞きました。次回ぜひ両方制覇したいです。」 |
| **山本** | 「ズンさん、大阪のディープな店も知っとるんやな!**お初天神**通りの**福太郎**、ぜひ。」 |

📝【1】 **広島お好み焼** = layered: bánh kếp mỏng → bắp cải → thịt → mì soba → trứng. 【2】 **大阪お好み焼** = mixed: trộn bột với cải + thịt + trứng → đổ chảo. 【3】 **そば入り** = có mì soba bên trong (đặc trưng Hiroshima).

> **VN:** Đứng giữa 2 vùng = ngoại giao. **Biết chi tiết kỹ thuật** (layer vs mix, soba) + tên quán cụ thể = chứng tỏ research nghiêm túc.

### 📍 Scenario 2 — 飛騨牛 + 朴葉味噌 (Gifu — anh Kato)

*dinner Gifu, 加藤さん dẫn đến quán Hida-Takayama · 自慢気 · càng vui*

| Speaker | Câu |
|---------|-----|
| **ズン** | 「加藤さん、本日はありがとうございます。**飛騨牛**【1】、ずっと食べたかったんです。」 |
| **加藤** | 「岐阜の宝やからな。**A5ランク**の霜降りは、口の中で溶けるよ。あ、これ**朴葉味噌**【2】、ご飯と一緒に食べてみて。」 |
| **ズン** | 「朴葉…葉っぱの上で味噌を焼くんですか?」 |
| **加藤** | 「そう、**朴の木**の葉。香りが移って、味噌に深みが出る。岐阜の山間部の郷土料理。」 |
| **ズン** | 「香り、すごいですね…ご飯何杯でもいけそうです。**中津川の栗きんとん**も有名って聞きました。」 |
| **加藤** | 「お、栗きんとんまで知っとるか。9月から11月の限定よ。**川上屋**の栗きんとん、東京駅でも買えるけど、現地で食べるのが一番。」 |
| **ズン** | 「秋にぜひ高山に行きたいです。**白川郷**もまだ見てなくて…」 |
| **加藤** | 「秋なら**飛騨高山祭**、4月と10月。屋台もすごい。一緒に行こうよ。」 |

📝【1】 **飛騨牛** = bò Hida, A5 grade marbled. 【2】 **朴葉味噌** (ほおばみそ) = miso nướng trên lá magnolia, đặc sản núi Gifu.

> **VN:** Khách Gifu 50t, đặc sản vùng = vốn quý không ai biết. Biết **朴葉味噌 / 中津川栗きんとん / 川上屋 / 飛騨高山祭** = level senior research.

### 📍 Scenario 3 — 名古屋 — 味噌文化 (Ito)

*Nagoya dinner · điềm tĩnh kiểu Nagoya · impress · cười nhẹ*

| Speaker | Câu |
|---------|-----|
| **ズン** | 「伊藤さん、名古屋飯は**赤味噌**文化って聞きましたが…」 |
| **伊藤** | 「そうですね。**八丁味噌**【1】が基本。味噌カツ、味噌煮込みうどん、土手煮…全部赤味噌。」 |
| **ズン** | 「**ひつまぶし**【2】も食べてみたいです。**蓬莱軒**って有名ですよね。」 |
| **伊藤** | 「お、よく勉強してますね。蓬莱軒の本店、神宮の近く。3食べ方ありますよね — そのまま、薬味、お茶漬け。」 |
| **ズン** | 「3パターン!まだ未経験です。**手羽先**は**世界の山ちゃん**派ですか、**風来坊**派ですか?」 |
| **伊藤** | 「あはは、これも戦争になる(笑)。私は風来坊派かな、甘めが好き。山ちゃんはガツン系。」 |
| **ズン** | 「両方食べ比べないと、ですね。あと**小倉トースト**って本当に喫茶店文化なんですか?」 |
| **伊藤** | 「名古屋の喫茶店、モーニング文化すごいよ。コーヒー1杯でトースト+ゆで卵+小倉あん。**コメダ珈琲**は東京にも進出したけどね。」 |

📝【1】 **八丁味噌** = miso đậu nành lên men 2-3 năm, đặc trưng Aichi. 【2】 **ひつまぶし** = lươn nướng cắt nhỏ trên cơm — 3 cách ăn.

> **VN:** Nagoya cuisine: **味噌カツ / ひつまぶし / 手羽先 / 小倉トースト / きしめん**. Biết tên quán **蓬莱軒 / 山ちゃん vs 風来坊 / コメダ** = lên rank.

### 📍 Scenario 4 — Variant: Hokkaido — biển + sữa (Nakamura)

*dinner Tokyo, Nakamura order · nostalgic*

| Speaker | Câu |
|---------|-----|
| **ズン** | 「中村CFO、北海道のおすすめ料理、何ですか?」 |
| **中村** | 「うーん、難しいな。海も山もあるからね。**海鮮**なら**ウニ**【1】**イクラ**【2】が圧倒的。**石狩鍋**は冬の定番。」 |
| **ズン** | 「**札幌ラーメン**は**味噌**ですよね。**すみれ**って有名って聞きました。」 |
| **中村** | 「すみれ、純連、けやき…どれも美味しい。あと**ジンギスカン**【3】**サッポロビール園**で食べると最高。」 |
| **ズン** | 「ジンギスカン、羊肉ですよね。北海道民は子供の頃から?」 |
| **中村** | 「BBQと言えばジンギスカン、運動会も町内会も。あと**ザンギ**【4】っていう唐揚げが地元ソウルフード。」 |
| **ズン** | 「ザンギ、初耳です!唐揚げと違いは?」 |
| **中村** | 「下味が濃い、ニンニク醤油生姜たっぷり。**ジンギスカンのキング**ってチェーン、東京にもあるから今度行ってみて。」 |

📝【1】 **ウニ** = nhím biển, Hokkaido top quality. 【2】 **イクラ** = trứng cá hồi muối. 【3】 **ジンギスカン** = thịt cừu nướng (Genghis Khan style), Hokkaido soul food. 【4】 **ザンギ** = gà chiên kiểu Hokkaido (hơi giống karaage nhưng đậm hơn).

> **VN:** Hokkaido: **海鮮 (ウニ・イクラ・カニ) / ラーメン味噌 / ジンギスカン / ザンギ / 石狩鍋**. Senior nostalgic — hỏi "子供の頃" = trúng tim.

## ⚠ NG — tuyệt đối tránh

- "ベトナム料理の方が美味しい" → KHÔNG so sánh tiêu cực.
- "[món] あまり好きじゃない" → khách host buồn. Chỉ nói "初めてです" / "新鮮です".
- Chê 1 vùng để khen vùng khác trước mặt khách vùng đó.
- Hỏi giá tiền món đắt (寿司 cao cấp / 神戸牛) → không lịch sự.
- Tự gọi món lạ trước khi check khách → mất quyền host.
