# Rule 20 — Tin tức quốc tế (an toàn) / 国際ニュース

> **Luận điểm.** Tin tức là **2-edge sword**: an toàn = **sport quốc tế** (WBC, World Cup, Olympics) + **thiên tai** (động đất, bão — đồng cảm). **CẤM TUYỆT ĐỐI**: chính trị Mỹ-Nhật-Trung, Bắc Triều Tiên, Đài Loan, scandal showbiz, vụ án hình sự cụ thể. Sai 1 câu = mất relationship.

---

## 🧠 Tâm lý JP (4 dòng)

- JP business culture **tránh tranh luận chính trị** trong work setting — quy tắc ngầm.
- Sport quốc tế (WBC, Olympics, World Cup) = topic universal an toàn.
- Khi có thiên tai trong nước JP → đồng cảm + chia sẻ KHÔNG phân tích → quan trọng.
- Đại sứ JP ở VN, business JP-VN → **OK** nhưng **không** đẩy comment so sánh chính trị.

---

## 🎬 Bối cảnh

Năm 2026 có WBC tháng 3 + bão lớn tháng 9 đổ bộ Kyushu. Dũng phải biết speak the language về sport + thiên tai chuẩn xác. Còn Linh ngây thơ hỏi chính trị Mỹ-Trung → bị nhắc nhở.

### 📍 Scenario 1 — WBC 2026: sport quốc tế (Hiroshi)

*meeting break, tháng 3 WBC đang diễn ra · impress*

| Speaker | Câu |
|---------|-----|
| **広島** | 「ズンさん、昨日の**WBC日韓戦**【1】、見た?」 |
| **ズン** | 「もちろん!**大谷翔平**【2】の**ホームラン**、しびれました。**佐々木朗希**【3】の100マイル超え球速も…」 |
| **広島** | 「サイドだね(笑)。**山本由伸**【4】の**フォークボール**、決勝でアメリカ相手にどうなるか。」 |
| **ズン** | 「アメリカは**マイク・トラウト**【5】が中心。**大谷vsトラウト**、再び見たいです、2023年の決勝みたいな…」 |
| **広島** | 「あれは**伝説の**最終打席【6】。三振取ったとき、日本中震えた。日本はベトナムでもWBC人気あると?」 |
| **ズン** | 「ベトナムは**ベースボール**マイナーですが、**MLBのDAZN中継**でファン増えてます。**ハノイ・ホーチミン**で**草野球**サークルもあります。」 |

> **VN:** Sport quốc tế: **WBC / Olympics / W杯 / NBA / MLB**. Trận **日韓** / **日米** = topic universal. Tên cầu thủ Nhật ở MLB (**大谷 / 山本 / 鈴木誠也 / 千賀**) = phải biết.

### 📍 Scenario 2 — 台風 (bão Kyushu 9月 — Sato bị ảnh hưởng)

*zoom buổi sáng sau bão lớn*

| Speaker | Câu |
|---------|-----|
| **ズン** | 「先生、お疲れ様です。**昨日の台風**【1】、福岡は大丈夫でしたか?」 |
| **佐藤** | 「ああズンさん、心配してくれてありがとう。うちは無事だったが、**博多湾**沿いの倉庫が一部水浸しでね。」 |
| **ズン** | 「ご無事で何よりです。**九州地方**【2】、毎年9月台風直撃ですよね。**避難所**【3】に行かれた方も多いとニュースで…」 |
| **佐藤** | 「今年は**最大瞬間風速65メートル**【4】、観測史上3番目だったとよ。**停電**【5】も2日続いた。」 |
| **ズン** | 「2日!ベトナムでも**中部地方**【6】で台風被害があって、皆さん大変です。**何か支援**できることあれば…」 |
| **佐藤** | 「ありがとう、その気持ちだけで嬉しか。**プロジェクト納期**は影響しないように頑張る。」 |
| **ズン** | 「**納期**は無理されず、まずお体とお仕事の安全を優先してください。**社内**にも伝えます。」 |
| **佐藤** | 「ベトナム側のサポート、本当に助かる。」 |

📝【1】 **台風** = bão. 【2】 **九州地方** = Kyushu region. 【3】 **避難所** = nơi sơ tán. 【4】 = 65m/s gió cực mạnh. 【5】 **停電** = mất điện. 【6】 **中部地方** = miền Trung VN.

> **VN:** Khi JP gặp thiên tai = **đồng cảm + đề nghị giúp** + **không** đẩy việc. Vocab: **台風 / 地震 / 津波 / 避難 / 停電 / 浸水 / 安否確認 / 復旧**. Senior cảm động khi VN partner care.

## ⚠ NG — tuyệt đối tránh

- Mở chính trị **米中 / 中台 / 韓国 / 北朝鮮**.
- Mention **天皇 / 靖国 / 戦争** — quá nhạy cảm.
- Comment scandal cá nhân tên tuổi (cả showbiz lẫn 政治家).
- "ベトナムは**社会主義**だから…" → tránh trigger.
- Đào sâu vụ án hình sự cụ thể đang trong news.
- So sánh chính trị Việt-Nhật-Trung → chìm thuyền.
