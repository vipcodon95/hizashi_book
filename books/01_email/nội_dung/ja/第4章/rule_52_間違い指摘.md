# Rule 52 — 間違いの指摘は質問の形で

> **要点.** 断定形の指摘（「その数字は違います」）は相手を*防御的*にする。質問の形で包む（「この数字は80万円でしょうか、800万円でしょうか」）と、相手が面目を保ちながら自己修正できる。
>
> キーワード：**「質問で指摘を包む」**。

## 悪い例

> 「松本様、数字が間違っています — こちらは80万円ではなく800万円です。」

→ 松本は「間違えた」と責められた感覚。否定や防御的反応の可能性。

## 良い例

> 「松本様、確認のためお伺いします — **この数字は80万円でしょうか、800万円でしょうか**？ 契約書に反映する前に確定させたく存じます。」

松本は面目を保って自己修正。800万円なら確認、80万円なら「ご指摘ありがとうございます」。

## 文型パターン

| 状況 | 質問形の指摘 |
|------|------------|
| 数字違い | 「この数字はXですか、Yですか？」 |
| 日付違い | 「この日付は木曜でしょうか、金曜でしょうか？」 |
| 名前違い | 「お名前はXでお間違いないでしょうか？」 |
| 論理矛盾 | 「私はA→Bと理解しております。解釈に相違はございませんでしょうか？」 |
| 大きな情報違い | 「再度拝見したところ、いくつか不明点がございます — ご確認いただけますでしょうか？」 |

## 直接言うべき場合

**安全性が面子より重要**な場合：
- セキュリティの誤り。
- 大きな影響のある金銭的誤り。
- 法的な誤り。

その場合：率直 ＋ 修正方法の提示 ＋ 責任追及なし。

## 使えるフレーズ

> **「小さな間違い → 質問形で。影響の大きい間違い → 率直だが解決策付きで。」**
