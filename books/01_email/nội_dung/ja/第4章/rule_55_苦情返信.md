# Rule 55 — 苦情への返信 — 感情の受容が先、解決策は後

> **要点.** 苦情メールへの**誤った反射**：直接解決策に入り、感情を無視すること。苦情を申し立てる人は**まず聞いてもらえた**と感じる必要があり、その後で解決策を受け入れる準備ができる。公式：**感情の受容 → 謝罪 → 原因 → 解決 → 再発防止**。
>
> **関連:** Rule 53（謝罪の4ステップ）は *自分から謝罪* するメール。Rule 55は *相手の苦情に応答* するメール — 冒頭の「感情の受容」が決定的な違い。
>
> キーワード：**「感情が先、解決策は後」**。

## 背景

白鷗社の松本PMから、ビンミンプロジェクトについての苦情メールが届いた。3回連続の納期遅延により、白鷗社の製品ローンチ計画に影響が出ている。ズン（弊社の担当者）が応答する場面。

## 悪い例 — 解決策直接

> 「松本様、納期遅延につきましてお詫びいたします。今後はバッファー2日を設定する新プロセスに切り替えます。5月分サービス料の5%補償を今週中に送金いたします。」

→ 松本は解決策で押し流された印象を受け、被った損害が認識されていないと感じる。むしろ怒りが増す可能性も。

## 良い例 — 5ステップ

> 「松本PM 様
>
> ご指摘のメール、しっかりと拝読いたしました。
>
> **【感情の受容】**
> 3回連続での納期遅延に対する松本様および貴社チームのご不満を、深く受け止めております。これは重大な問題であり、貴社の製品ローンチ計画 — 自由に動かせない重要なマイルストーンに直接影響を与えたものと認識しております。決して軽く考えておりません。
>
> **【謝罪】**
> ティエンファットチームを代表し、松本様および貴社チームの皆様に、心よりお詫び申し上げます。
>
> **【原因】**
> 社内調査の結果、3つの原因を確認いたしました：
> - 納品物1（4/29）：弊社のリソース調整ミス — 全面的に弊社の責任。
> - 納品物2（5/07）：5日遅延のうち、トゥアンリーダーの病欠で2日、バックアッププラン不足で3日。
> - 納品物3（5/14）：新QAプロセスが想定より長引き4日遅延。
>
> **【解決策】**
> - 契約に基づく遅延損害金：5月分サービス料の10%（80万円）、5/20までにロアン経理部長より送金手続き。
> - 善意の表明として6月分サービス料を5%割引。
> - 貴社のローンチスケジュール挽回のため、フェーズ2を1週間前倒し。
>
> **【再発防止】**
> - フオン副部長を白鷗社専属アカウントマネージャーに指定済み。
> - 6月以降、全納品物コミットに2日のバッファーを設定。
> - 月曜09:00に進捗レポートを毎週送付（休週なし）。
>
> 来週、貴社オフィスに直接お伺いし、詳細計画のご説明とご要望のヒアリングをさせていただきたく存じます。5/19〜22の間でご都合をお知らせください。
>
> 改めまして、心よりお詫び申し上げます。
>
> ズン
> ティエンファット社」

→ 松本は解決策を読む前に「ティエンファット社が問題の重大性を理解した」と感じる → 解決策を受け入れる素地ができる。

## Rule 53 と Rule 55 の比較

| | Rule 53 — 自分から謝罪 | Rule 55 — 苦情への応答 |
|--|------------------|---------------------|
| 状況 | 自分から問題を報告 | 相手から既に苦情が来た |
| ステップ数 | 4（事実→原因→対策→約束） | 5（感情の受容→謝罪→原因→解決→再発防止） |
| 追加ステップ | — | 「感情の受容」を冒頭に — 既に怒っている相手をまず受け止める |
| トーン | 認知＋約束 | 傾聴＋約束 |

## 使えるフレーズ

> **「感情の受容 → 謝罪 → 原因 → 解決 → 再発防止。5ステップ、特に最初の1ステップを省かない。」**
