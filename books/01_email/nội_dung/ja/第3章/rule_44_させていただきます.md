# Rule 44 — 「させていただきます」は慎重に — 濫用を避ける

> **要点.** 「〜させていただきます」は近年最も誤用される敬語の一つ。本来は**相手の許可や恩恵に依る場合**に使う謙譲表現。だが現在、あらゆる動作に付ける人が多く、耳障りかつ不自然になっている。**「いたします」で済む場面で「させていただきます」を使わない。**
>
> キーワード：**「させていただきます は限定的に」**。

## 正しい用法

「させていただきます」を使うのは **2つの条件を満たす時のみ**：
1. 相手の**許可・許諾**を得ている、または必要な場合。
2. その行為で自分に**恩恵**がある場合。

## 悪い例

> 「お送りさせていただきます。」
> → 送るのは自分の動作で、相手の許可は関係ない。「お送りいたします」で十分。
>
> 「確認させていただきます。」
> → 同様、「確認いたします」で十分。
>
> 「出社させていただきます。」（通常の業務報告として）
> → 出社は通常の業務、許可の概念なし。「出社いたします」で十分。
>
> *例外：* 上司から「明日休んでいいよ」と提案された後で、それでも出る場合は「出社させていただきます」が自然（相手の配慮を踏まえた行為のため）。

## 良い例

> 「貴社の資料を**参考にさせていただきます**。」
> → 相手の資料を使う許可的な要素あり。
>
> 「研修に**参加させていただきます**。」
> → 研修は相手から提供されたもの、許諾と恩恵がある。
>
> 「この件については**辞退させていただきます**。」
> → 通常の「辞退いたします」より柔らかく、相手への配慮を込める。

## 「いたします」で十分な場面

| 動作 | 推奨 |
|------|------|
| 送信 | お送りいたします |
| 確認 | 確認いたします |
| 連絡 | ご連絡いたします |
| 報告 | ご報告いたします |
| 提出 | ご提出いたします |
| 回答 | ご回答いたします |

## 使えるフレーズ

> **「許可・恩恵が絡むなら『させていただきます』。そうでないなら『いたします』で十分。」**
