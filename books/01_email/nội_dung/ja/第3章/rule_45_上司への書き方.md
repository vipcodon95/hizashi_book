# Rule 45 — 上司宛 — 指示調を避け、依頼形にする

> **要点.** 日本語には*指示調・命令調*の響きを持つ表現がある。上司や目上の方への文では、これらを**依頼形・相談形**に変える必要がある。一語の違い（「ください」→「いただけますか」）が敬意の温度を大きく変える。
>
> キーワード：**「依頼形、指示形を避ける」**。

## 対照表

| 指示調（上司にはNG） | 依頼形（上司に適切） |
|--------------------|------------------|
| 「木曜までに承認してください」 | 「木曜までにご承認いただけますでしょうか」 |
| 「この点を明確にしてください」 | 「この点につきまして、お教えいただけますでしょうか」 |
| 「添付をご覧ください」 | 「添付をご参照いただければ幸いです」 |
| 「知っておいてください」 | 「ご認識いただければと存じます」 |
| 「理解してください」 | 「ご容赦いただけますと幸いです」 |
| 「すぐ対応してください」 | 「お手すきの際にご対応いただけますと幸いです、できれば金曜までに」 |

## 悪い例

> 「フオン副部長、50万円の追加予算を木曜までに承認してください。承認結果を私に返送してください。」

→ 指示調 — 部下や目下には問題ないが、上司には不適切。

## 良い例

> 「フオン副部長、ビンミンの追加予算50万円につきまして、5月15日（木）までにご承認いただけますと幸いです。ご承認後、結果をお知らせいただければ、次の工程に進めます。
>
> よろしくお願い申し上げます。」

同じ依頼、正しい語調。

## 明確さは維持、卑屈にならない

依頼形 ≠ 卑屈。必須項目は明確に：
- 期限。
- 必要な行動。
- 遅延時の影響（該当する場合）。

## 使えるフレーズ

> **「上司宛：『〜いただけますでしょうか』『〜いただけますと幸いです』。『〜してください』は避ける。」**
