# Rule 37 — 「了解」は社内だけ、社外・上司には「承知」

> **要点.** ビジネス日本語で最もよく誤用されるのが「了解しました」。これは**対等または目下**に使う表現で、上司・顧客・取引先には**不適切**。正しくは「**承知いたしました**」または「**かしこまりました**」。1文字の違いだが、相手への敬意の有無を示す重要なシグナル。
>
> キーワード：**「了解 vs 承知」**。

## 使い分け表

| 相手 | 推奨 |
|------|------|
| 対等の同僚、親しい同世代 | 了解しました／了解です |
| 部下 | 了解しました |
| 上司 | **承知いたしました** |
| 社外（取引先・顧客） | **承知いたしました／かしこまりました** |
| 大口顧客・経営層 | **かしこまりました** |

## 悪い例

> 松本PM（白鷗社、顧客）からのメールに：
> 「了解しました、対応いたします。」

→ 松本は*違和感*を覚える。「自分はズンさんの目下扱いか？」という印象。

## 良い例

> 松本様からのメールに：
> 「承知いたしました、対応いたします。」

## 類似の誤用

| 不適切 | 適切 |
|-------|------|
| 了解しました（上司・社外へ） | 承知いたしました |
| ご苦労様です（目上へ） | お疲れ様です |
| なるほど（上司へ） | おっしゃる通りです／さようでございますね |
| 参考になりました | 勉強になりました |

## 使えるフレーズ

> **「了解 は対等・目下に、承知 は上司・社外に。1文字の違いが敬意の有無を表す。」**
