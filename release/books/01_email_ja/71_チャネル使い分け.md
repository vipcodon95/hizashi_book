# Rule 70 — ChatツールとEmailの使い分け — 何を代替し、何は代替できないか
> **要点.** チャットツール（Teams、Slack、LINE WORKS）は日常業務の多くでメールより速い。しかし *証跡* が必要な場面、*体系的な保管* が必要な場面、*フォーマル* が必要な場面では**メールの代替にならない**。正しいチャネル選択が大きな時間節約になる。
>
> キーワード：**「Chat vs Email」**。

## 比較表
| 場面 | 推奨 |
|------|------|
| 同日の簡単な質問 | **Chat** |
| 1時間以内の打ち合わせ | **Chat**（ビデオ通話） |
| ブレインストーミング | **Chat** |
| 軽いファイル送付（社内） | **Chat** |
| **契約書・正式見積** | **Email** |
| **顧客への謝罪メール** | **Email**（敬意） |
| **ポリシー変更通知** | **Email**（保管必要） |
| **初回の顧客連絡** | **Email** |
| **週報・月報** | **Email** |
| **部下への正式な業務指示** | **Email**（責任証跡） |
| **後で検索が必要な情報** | **Email**（検索しやすい） |

## チャネル切替のサイン
### Chat → Email
- 事項が思ったより重要になった。
- CCして他者にフォローしてほしい。
- 後で公式記録が必要になる可能性。
- 大きなファイルと多くの注記。

### Email → Chat
- メール往復3回でも噛み合わない（Rule 19）。
- 事項が急に緊急度を増した。
- 1行の短い回答で十分。

## 使えるフレーズ
> **「Chatはスピード、Emailは保管。正しく選べば週2時間節約。」**
