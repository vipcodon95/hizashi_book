# Rule 62 — 漢字を使いすぎない、ひらがなで読みやすく
> **要点.** 連続する漢字の塊は視覚的に重く、読み疲れを誘う。同じ意味でも**ひらがなに開く**ことで軽くなる場合が多い。ただし専門用語や公式文書では適切な漢字使用が望ましい — 場面に応じた判断が必要。
>
> キーワード：**「漢字を開く」**。

## 漢字→ひらがな対照例
| 漢字表記（重い） | ひらがな（軽い） |
|----------------|--------------|
| 出来る／出来ません | できる／できません |
| 致します | いたします |
| 宜しく | よろしく |
| 於いて | において |
| 様々 | さまざま |
| 事／物／時 | こと／もの／とき |
| 下さい | ください |
| 御礼／御願い | お礼／お願い |
| 有難う | ありがとう |
| 何卒 | ※正式な場面では漢字も可 |

## 悪い例（漢字過多）
> 「此度の件に於きましては、下記事項を宜しく御願い致します。」

## 良い例（バランス）
> 「この度の件につきまして、下記事項をよろしくお願いいたします。」

同じ内容、読みやすさが大幅向上。

## 漢字を使うべき場面
- 専門用語（法務、金融）。
- 公式文書（契約書、公式レター）。
- ひらがなが見た目に幼稚すぎる文脈。

## 使えるフレーズ
> **「ひらがなで書ける事は、ひらがなで。メールが軽く、読みやすくなる。」**
