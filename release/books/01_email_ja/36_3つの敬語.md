# Rule 36 — 尊敬語・謙譲語・丁寧語 — 3つの使い分け
> **要点.** 日本語敬語は**3種類**：(1) **尊敬語** — 相手の動作を高める（いらっしゃる、お越しになる）、(2) **謙譲語** — 自分の動作をへりくだる（伺う、申し上げる）、(3) **丁寧語** — 全体を丁寧にする（です・ます）。社内外、上下関係によって適切な組み合わせが変わる。
>
> キーワード：**「3種敬語の使い分け」**。

## 動詞別の対応表（よく使うもの）
| 普通形 | 尊敬語（相手） | 謙譲語（自分） |
|-------|-------------|-------------|
| 行く／来る | いらっしゃる／お越しになる | 伺う／参る |
| 言う | おっしゃる | 申す／申し上げる |
| 食べる／飲む | 召し上がる | いただく |
| 見る | ご覧になる | 拝見する |
| 知る／知っている | ご存じ（です） | 存じる／存じ上げる |
| する | なさる | いたす |
| 受け取る | お受け取りになる | 頂戴する |
| 与える／くれる | くださる | 差し上げる |
| 読む | お読みになる | 拝読する |

> 注：「もらう」自体は元々謙譲側の動詞（「いただく／頂戴する」）。相手の「受け取る」動作を尊敬する場合は「お受け取りになる」を使う。「くれる／与える」の尊敬語は「くださる」。混同しないこと。

## 悪い例 — 混用
> 「松本様、資料を**拝見してくださいましたか**？」

「拝見する」は自分の動作（謙譲語）。相手に使うなら「ご覧になる」「ご覧いただく」。

## 良い例
> 「松本様、資料を**ご覧いただけましたでしょうか**？」

## 使えるフレーズ — 5つのコア
1. 「伺います」（自分が行く）
2. 「おっしゃる通りです」（相手の言）
3. 「拝見いたしました」（自分が見た）
4. 「ご確認いただけますでしょうか」（相手に確認を依頼）
5. 「かしこまりました」（承知の意）

## 核心原則
> **「相手の動作 → 尊敬語。自分の動作 → 謙譲語。迷ったら『お〜する／させていただく』より『〜いたします』。」**
