# Rule 37 — Khí hậu 3 miền VN / 3地域気候
> **Luận điểm.** "VN nóng quanh năm" = nhận thức sai phổ biến của khách Nhật. **Hà Nội có 4 mùa thật**, **Đà Nẵng-Huế hứng bão và lũ tháng 10**, **HCM 2 mùa mưa-khô**. Bạn kể được 3 miền khí hậu = khách Nhật chuẩn bị quần áo onsite đúng + có topic sâu để bridge với 4 mùa JP.

---

## Tâm lý JP (4 dòng)
- JP văn hóa 二十四節気 — họ rất quan tâm thời tiết và sẽ check trước onsite.
- "Hà Nội mùa đông 5°C" gây bất ngờ thực sự — JP tưởng tropical 365 ngày.
- "Bão miền Trung tháng 10" = liên tưởng 台風 quê họ → bridge instant về 防災.
- HCM "2 mùa" (mưa-khô) tương đối lạ với JP — cần explain được "mưa 30 phút rồi tạnh" pattern.

---

## 4 Scenarios — dialogue thực chiến
### Scenario 1 — Hà Nội 4 mùa thật
*意外*

| Speaker | Câu |
|---------|-----|
| **松本** | 「ズンさん、来年1月にハノイ出張なんだけど、半袖でいいよね?」 |
| **ズン** | 「いえ、**ハノイは1月の寒波で10度切ることもあります**。**ダウンジャケット要ります**。」 |
| **松本** | 「ええっ、ベトナム10度?」 |
| **ズン** | 「ハノイは**4季節あって**、12-2月の冬は東京並みに冷えます。**ただ暖房がない家が多い**ので、屋内も寒く感じます。」 |
| **松本** | 「暖房ないの?それは辛そう…」 |
| **ズン** | 「室内で**ヒートテック+セーター**で過ごします。あと**北部独特の『湿った冷気』**で、東京の乾いた冬と違って体に染みる感じです。」 |
| **松本** | 「分かった、防寒しっかりしていく。逆に夏は?」 |
| **ズン** | 「夏は逆で、**夏は35度を超える日が多く、稀に40度近くまで上がります**。日本の蒸し暑さと近いです。**春(3-4月)と秋(10-11月)が一番過ごしやすい**ので、出張ベストシーズンです。」 |

> **VN:** Phá định kiến "VN nóng quanh năm" → khách Nhật chuẩn bị quần áo đúng + thấy người VN có depth.

### Scenario 2 — Bão và lũ miền Trung
*Okinawa · check ngay weather*

| Speaker | Câu |
|---------|-----|
| **小林** | 「ダナンの出張、来週にしようか今週にしようか迷ってて。今って大丈夫?」 |
| **ズン** | 「**今週は台風12号**が南シナ海に発生したばかりで、**ダナンに金曜接近**の予報です。来週の方が確実に安全です。」 |
| **小林** | 「あー、台風?沖縄もしょっちゅう来るから慣れてるけど、ベトナムも来るんだ。」 |
| **ズン** | 「**中部(フエ・ダナン・ホイアン)が一番台風に弱い**地域で、**9-11月は台風シーズン**。2017年のダムレイ台風では、ホイアン旧市街が**腰の高さまで浸水**しました。」 |
| **小林** | 「腰までか…沖縄の台風被害と似てるな。」 |
| **ズン** | 「沖縄と中部VNはほぼ同じ緯度なので、台風の通り道がよく似てます。**現地は旧市街の家を1階空っぽにして、家具を2階に上げる**【1】っていう適応してます。」 |
| **小林** | 「そりゃ生活の知恵だね。じゃ来週で調整しよう。」 |

📝【1】 Hội An, Huế: "ngôi nhà 1 tầng để trống mùa lũ" — chi tiết vivid + practical dùng được. Sẽ ấn tượng khách Okinawa quen 防災.

### Scenario 3 — HCM 2 mùa mưa khô
| Speaker | Câu |
|---------|-----|
| **山本** | 「来月ホーチミン、雨季だよね?ずっと降ってんの?」 |
| **ズン** | 「**降り続ける雨**ではなく、**一日1〜2回、強く30分降ってサッと止む**スコール型です。」 |
| **山本** | 「あー、シンガポールみたいな?」 |
| **ズン** | 「まさにそれです!**典型的な熱帯モンスーン**で、**5-10月が雨季、11-4月が乾季**。雨季は**夕方4-6時**に降ることが多いので、**その時間帯は屋内**で予定組むのがコツです。」 |
| **山本** | 「夕方かあ、ちょうど移動の時間や。」 |
| **ズン** | 「はい、なので**12時頃に外回り、夕方に会議室**っていう逆転パターンが現地BDの定番です。あと**雨が止んだ直後は涼しくて気持ちいい**。」 |
| **山本** | 「現地の人の動き方ってのが分かるね、メモる。気温は?」 |
| **ズン** | 「**ホーチミンは年間27-32度ほぼ一定**で、ハノイみたいな温度差はないです。**11-1月の乾季・朝晩涼しめ**が、出張ベストシーズン。」 |

> **VN:** Pattern "12時外回り / 夕方屋内" = kiến thức local insider khách JP cực thích — vì it actionable.

### Scenario 4 — So sánh với 4 mùa Nhật + bridge
| Speaker | Câu |
|---------|-----|
| **松本** | 「日本の四季文化って、ベトナムの人にはピンと来ない?」 |
| **ズン** | 「**ハノイ出身者には四季は分かります**。私も小さい頃**桃の花の春、ロータスの夏、菊の秋、菊酒の冬**って母から教わって育ちました。」 |
| **松本** | 「あ、ベトナムにも季語みたいなのあるんだ。」 |
| **ズン** | 「**北部の伝統文化には四季感**ありますね。**ホーチミン出身者には逆にピンと来ない**ようで、彼らに『紅葉狩り』の話をしても初めて知ることが多いです。」 |
| **松本** | 「面白いね、同じ国でも違うんだ。」 |
| **ズン** | 「だから**北の人と話す時は『四季の話』、南の人と話す時は『マンゴーの旬』『熱帯果実の話』**って、topic変える必要があります。」 |
| **松本** | 「なるほど、東京と那覇くらい文化違うのか。」 |
| **ズン** | 「まさにそういうイメージです、近いと思います!」 |

> **VN:** "Hà Nội ↔ HCM = Tokyo ↔ Naha" bridge clever cho khách Okinawa-quan-tâm scene.

## NG — tuyệt đối tránh
- "VN nóng quanh năm, dễ chịu" → sai, gây khách mặc sai onsite tháng 1 HN.
- Bão miền Trung → mention Aid / chính trị quốc gia → vùng cấm.
- "JP có 4 mùa thật, VN không có" → arrogance, sai (HN có).
- Khoe "VN có cái Nhật không có" hoặc ngược lại → so sánh negative.
- Không cảnh báo trước về bão khi khách lên kế hoạch onsite tháng 9-11 → trách nhiệm.
