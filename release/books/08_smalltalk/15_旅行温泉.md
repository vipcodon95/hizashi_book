# Rule 15 — Du lịch + onsen / 旅行・温泉
> **Luận điểm.** 旅行 = topic thân khi đã quen 3-6 tháng. **温泉** đặc biệt — JP nhìn ai cũng thấy "nguồn an ủi". Biết phân biệt **旅館 vs ホテル**, **会席 vs 懐石**, top 10 onsen ranking, etiquette tắm onsen = bạn đẳng cấp host.

---

## Tâm lý JP (4 dòng)
- 温泉 = **văn hóa healing** đẳng cấp #1 Nhật. Senior 50-60t coi onsen weekend = thuốc thần.
- Mỗi vùng có 名湯 riêng — Hokkaido **登別** / Kyushu **黒川** / Gunma **草津** / Hyogo **有馬**.
- 旅館 ≠ ホテル. Ngủ futon trên tatami + 浴衣 + kaiseki = trải nghiệm văn hóa.
- Etiquette tắm: **tắm rửa kỹ trước**, **tóc búi cao**, **không khăn vào nước**, **xăm bị cấm 1 số nơi** → bạn **không** có xăm thì OK.

---

## Bối cảnh
Dũng + Hà CTO sắp đi business trip combine với 1 đêm ở 旅館 tại 箱根. Ōgaki gợi ý đặt giúp. Đồng thời Nakamura kể về 登別 (Hokkaido) và Sato kể về 黒川 (Kumamoto). Dũng phải biết speak the language của onsen thế giới.

## 4 Scenarios — dialogue thực chiến
### Scenario 1 — 箱根 — gợi ý 旅館 + kaiseki
*meeting break*

| Speaker | Câu |
|---------|-----|
| **大垣** | 「ズンさん、来週**箱根**【1】に行くんでしょ?**旅館**【2】はもう決めた?」 |
| **ズン** | 「まだなんです…おすすめあります?」 |
| **大垣** | 「**強羅**【3】エリアの**箱根強羅花壇**か、**仙石原**の**ふふ箱根**、**会席料理**【4】が美味しい。**露天風呂付き客室**【5】、ある?」 |
| **ズン** | 「初体験なので、ぜひ**露天風呂付き**でお願いしたいです。**会席と懐石**【6】の違いって何ですか?」 |
| **大垣** | 「会席=お酒を楽しむコース料理。懐石=茶事の前の軽食、もっと侘び寂び。旅館は会席が多い。**先付・前菜・お造り・煮物・焼き物・揚げ物・蒸し物・酢の物・ご飯・止め椀・水菓子**【7】、10品くらい。」 |
| **ズン** | 「10品!驚きですね。**仲居さん**【8】が部屋まで運んでくれるんですよね?」 |
| **大垣** | 「そう、部屋食。あと**チェックイン後すぐ温泉**【9】、**夕食前にもう1回**、**朝起きてもう1回**、これが旅館の楽しみ方。」 |
| **ズン** | 「3回!1泊で3回入るんですね。覚えておきます。」 |

📝【1】 **箱根** = onsen mecca gần Tokyo. 【2】 **旅館** = ryokan (kiểu truyền thống). 【3】 **強羅・仙石原・湯本** = 3 vùng onsen lớn Hakone. 【4】 **会席料理** = course đa món kèm rượu. 【5】 **露天風呂付き客室** = phòng có bồn lộ thiên riêng (premium). 【6】 **懐石** = light meal pre-茶事 (formal). 【7】 = 10 món truyền thống. 【8】 **仲居** = nakai-san (女性 phục vụ). 【9】 Vào onsen 2-3 lần/đêm = chuẩn.

> **VN:** Vocab onsen-ryokan: **会席 / 懐石 / 露天風呂 / 仲居 / 部屋食 / 朝風呂 / 夕食前湯 / 浴衣**. Senior cảm động khi người Việt biết đúng văn hóa 1 đêm 旅館.

### Scenario 2 — 登別: Hokkaido + 地獄谷 (Nakamura kể)
*zoom kể chuyện cuối tuần · impress*

| Speaker | Câu |
|---------|-----|
| **中村** | 「先週末、**登別**【1】行ってきたよ。**地獄谷**【2】、本当に湯気が上がってて、別世界。」 |
| **ズン** | 「登別!**第一滝本館**【3】は有名ですよね。**5種類の泉質**【4】がある宿って聞きました。」 |
| **中村** | 「滝本館まで知ってるんだ!**硫黄泉**【5】**鉄泉**【6】**酸性泉****重曹泉**…全部体に違う効能。**白い濁り湯**は美肌、**茶色い鉄泉**は冷え性。」 |
| **ズン** | 「泉質ごとに効能違うんですね。**温泉ソムリエ**って資格、本当にあるんですか?」 |
| **中村** | 「あるよ(笑)。週末で取れる入門編。北海道は**洞爺湖****ニセコ****定山渓**もいいよ。**冬は雪見露天**、最高。」 |
| **ズン** | 「雪見露天!夢ですね。**外気温-5度**で湯に浸かるって…」 |
| **中村** | 「肩から上は冷たいけど、湯の中は42度。**毛細血管が広がって**、めちゃくちゃ温まる。来年2月、雪まつりと一緒にどう?」 |
| **ズン** | 「ぜひ計画させてください!」 |

📝【1】 **登別温泉** (Noboribetsu) = top onsen Hokkaido. 【2】 **地獄谷** = "thung lũng địa ngục" — hơi nước nóng. 【3】 **第一滝本館** = ryokan 150 năm. 【4】 = 5 loại nước khác nhau cùng resort. 【5】**硫黄泉** = lưu huỳnh. 【6】**鉄泉** = sắt.

> **VN:** Mỗi onsen có **泉質** riêng. **硫黄泉 / 鉄泉 / 単純泉 / 塩化物泉 / 炭酸水素塩泉**. Vocab này = senior level.

### Scenario 3 — 黒川温泉: Kyushu (Sato giới thiệu)
*rạng rỡ*

| Speaker | Câu |
|---------|-----|
| **佐藤** | 「ズンさん、九州来たら**黒川温泉**【1】、絶対行ってほしいとよ。」 |
| **ズン** | 「黒川!**入湯手形**【2】で**3つの旅館**の**露天風呂**を巡れるんですよね?」 |
| **佐藤** | 「お、知っとるか!**1300円**で3軒、6か月有効。村全体が一つの旅館って思想で、街並みも統一感ある。」 |
| **ズン** | 「街全体が…素敵な思想ですね。**新明館**の**洞窟風呂**、**山みず木**の**渓流露天**、有名って聞きました。」 |
| **佐藤** | 「全部押さえてる(笑)。あと**いこい旅館**の**滝の湯**もよかとよ。**阿蘇山**【3】とセットで行くのがおすすめ。**湯布院**【4】も近いから2泊で回れる。」 |
| **ズン** | 「湯布院!**金鱗湖**の朝霧、写真で見て憧れました。**由布岳**【5】見ながらの露天、最高でしょうね。」 |
| **佐藤** | 「ベトナムから九州、**福岡空港**直行便あるとよ。次回出張時に1日伸ばせば十分回れる。**馬刺し**【6】**地鶏**【7】も食べないと。」 |
| **ズン** | 「黒川+湯布院+阿蘇、絶対計画します!」 |

📝【1】 **黒川温泉** (Kumamoto) = onsen làng truyền thống. 【2】 **入湯手形** = "thẻ tắm" gỗ — vào 3 ryokan khác nhau. 【3】 **阿蘇山** = núi lửa Aso. 【4】 **湯布院** (Oita) = onsen khác Kyushu. 【5】 **由布岳** = núi Yufu. 【6】 **馬刺し** = sashimi ngựa Kumamoto. 【7】 **地鶏** = gà địa phương Miyazaki.

> **VN:** **湯布院** vs **黒川** vs **別府** = 3 onsen Kyushu. Mỗi nơi có concept riêng. **入湯手形** ở Kurokawa = đặc sản trải nghiệm.

### Scenario 4 — Etiquette onsen (lần đầu Dũng đi)
*lần đầu, hỏi 大垣 trước khi đi*

| Speaker | Câu |
|---------|-----|
| **ズン** | 「大垣さん、温泉初めてで…**マナー**で気をつけることありますか?」 |
| **大垣** | 「いい質問。まず**かけ湯**【1】、入る前に必ず**お湯を体にかけて**慣らす。あと**洗い場で先に体を洗う**、**シャンプー残さない**。」 |
| **ズン** | 「**タオル**は湯船に入れちゃダメですよね?」 |
| **大垣** | 「そう、**タオルは湯船の縁か頭の上**。あと**髪が長い人は結ぶ**。湯に髪が浸からないように。」 |
| **ズン** | 「**入れ墨**【2】、ベトナム人だと友達でアームに小さく入れてる人いるんですが…」 |
| **大垣** | 「最近は**タトゥー可**の宿も増えてるけど、**伝統的な旅館はNG**多い。事前にチェック必要。**シール**で隠せる**カバー**もある。」 |
| **ズン** | 「気をつけます。あと**写真撮影**は当然NG?」 |
| **大垣** | 「**脱衣所**でも**浴室**でもNG。**廊下**と**外**だけ。あと**飲酒後すぐ温泉**は**気をつけて**、血圧上がるから。**水分補給**忘れずに。」 |

📝【1】 **かけ湯** = dội nước trước khi vào bồn. 【2】 **入れ墨** (irezumi) = xăm. Truyền thống = tránh.

> **VN:** Etiquette onsen 7 quy tắc: ① **かけ湯** trước; ② **rửa kỹ ở 洗い場**; ③ **tóc búi cao**; ④ **không tay khăn vào nước**; ⑤ **không xăm** (truyền thống); ⑥ **không chụp ảnh** trong khu tắm; ⑦ **không vào ngay sau rượu**.

## NG — tuyệt đối tránh
- Vào 湯船 mà **không かけ湯** trước → bị other guests trừng mắt.
- **Mang tay khăn xuống nước** → vi phạm rule cơ bản.
- **Chụp ảnh** trong 脱衣所 / 浴室 → có thể bị báo cảnh sát.
- Khoe khoang đã đi onsen X, Y, Z → kể đặc sản từng nơi mới có giá trị.
- Hỏi giá phòng 旅館 cao cấp → không lịch sự.
