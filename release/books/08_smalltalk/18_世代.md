# Rule 18 — Generation references / 世代の話題
> **Luận điểm.** JP có **3 thế hệ rõ rệt**: **昭和** (1926-1989, ký ức tuổi thơ + バブル) / **平成** (1989-2019, デフレ世代) / **令和** (2019-, デジタルネイティブ). Match đúng reference với tuổi khách = bạn nói cùng ngôn ngữ. Sai = lệch.

---

## Tâm lý JP (4 dòng)
- Thế hệ JP đo bằng **元号** (era 元号 thay vì năm dương lịch).
- 50-60t = 昭和世代, **nostalgic** về 1970-80s tuổi thơ + バブル経済 1986-1991.
- 30-40t = 平成世代, biết **デフレ** + **失われた10年**. Hobby anime/game OK với họ.
- 20-30t = 令和世代 / Z世代, **TikTok / SNS / 推し活**. Hỏi "Showa nào?" = họ KHÔNG biết.

---

## Bối cảnh
Dũng phải xoay reference theo từng đối tác: Sato 60t (Showa nostalgia), Hiroshi 50t (バブル), Tanaka 35t (平成 game/anime), Linh 22t (令和). Mỗi thế hệ có culture references hoàn toàn khác.

## NG — tuyệt đối tránh
- Hỏi senior 60t về **TikTok / Vtuber / 推し** → trống không.
- Hỏi 22t về **山口百恵 / バブル** → họ KHÔNG biết, lệch awkward.
- "**昭和の人だね**" trêu senior → trêu này JP **không** OK.
- So sánh "**令和の若者は楽しすぎる**" → tránh comment thế hệ tiêu cực.
- Mất khả năng catch khi senior nói **元号** (昭和60年 = 1985) → cần quy đổi nhanh.

## BJT
**J3 — 場面把握:** Match age → era + reference. Senior 60t → Showa nostalgia.
**J2 — 表現読解:** Phân biệt "**昭和の人**" (nostalgic) vs "**昭和的**" (cũ kỹ).
**J1 — 発言聴解:** Quy đổi 元号 → dương lịch. 平成3年 = 1991. 令和8年 = 2026.
