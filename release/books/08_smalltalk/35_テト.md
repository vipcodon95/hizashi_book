# Rule 35 — Tết — solar vs lunar / テト・旧正月
> **Luận điểm.** Khách Nhật biết "Tết" nhưng hay nhầm với 正月 (1/1). Bạn phải kể được: **âm lịch khác dương lịch, 12 con giáp giống JP nhưng 1 con khác (Trâu vs Ushi/Bò), hoa đào Bắc / hoa mai Nam, lì xì, mâm ngũ quả** — và **tránh cõi chính trị** quanh Tết.

---

## Tâm lý JP (4 dòng)
- Người Nhật từ Meiji đã chuyển 正月 sang dương lịch — họ tò mò khi VN vẫn giữ âm lịch.
- 12 con giáp gần như giống y JP — chỉ **Trâu/Bò (丑) và Mèo/Thỏ (卯/兎)** khác → đây là điểm "ơ thật á?" mà JP cực thích.
- JP rất thích món truyền thống có **lý nghĩa tinh thần** (おせち料理) → mâm ngũ quả, bánh chưng cùng phạm trù.
- TUYỆT ĐỐI tránh: Tết Mậu Thân, đoàn tụ Bắc-Nam, dòng người về quê = vùng chính trị.

---

## Bối cảnh
Cuối tháng 1/2026 (gần Tết VN — mùng 1 = 17/2/2026, năm Ngọ 午年). Dũng zoom call với Matsumoto / Kato (Gifu) chuẩn bị schedule Phase 5 — phải explain "VN nghỉ Tết 1 tuần" mà khách Nhật chưa hiểu cơ chế.

## 4 Scenarios — dialogue thực chiến
### Scenario 1 — Explain mechanism âm vs dương lịch
| Speaker | Câu |
|---------|-----|
| **松本** | 「ズンさん、2月の半ばに連休あるってメールにあったけど、何の祝日?」 |
| **ズン** | 「ベトナムのテト(旧正月)です。**旧暦の1月1日**で、毎年日付が変わります。今年は2月17日、来年は2月6日です。」 |
| **松本** | 「あー、中国の春節と同じタイミングだね。」 |
| **ズン** | 「はい、東アジアで旧正月を祝う国はベトナム、中国、韓国(ソルラル)、シンガポールなど。**日本だけ明治期に新暦に切り替えた**んですよね。」 |
| **松本** | 「そうそう、明治5年だっけ。じゃあベトナムは正月を年2回?」 |
| **ズン** | 「形式上はそうですが、**新暦の1月1日は1日だけ祝日**で軽め。**テトが本番**です。1週間〜10日休みで、地方に帰省して家族と過ごします。」 |
| **松本** | 「日本のお盆+正月を一緒にした感じ?」 |
| **ズン** | 「まさにそのイメージです!帰省ラッシュも激しくて、新幹線にあたる**南北統一鉄道**が満員になります。」 |

> **VN:** Bridge "Tết = お盆 + 正月" → khách Nhật instant hiểu scale + tình cảm.

### Scenario 2 — 12 con giáp + năm Ngọ 2026
*Gifu, có vợ thuộc 1986 寅年*

| Speaker | Câu |
|---------|-----|
| **加藤** | 「ズンさん、ベトナムも干支あるんでしょ?」 |
| **ズン** | 「はい、**ほぼ日本と同じ12支**ですが、4つ違いがあります、特に大きく違うのは2つで、**牛の代わりに水牛**、**兎の代わりに猫**【1】です。」 |
| **加藤** | 「ええっ、猫?可愛いね!何でだろう?」 |
| **ズン** | 「諸説あるんですが、**ベトナム語で『卯(マオ)』が猫(メオ)に音が似てる**から、と聞いたことがあります。」 |
| **加藤** | 「面白いなあ、音の混同ね。じゃあ今年は何年?」 |
| **ズン** | 「**午年(うま)**です、これは日本と同じ。私は1995年生まれの**亥年(いのしし)**ですが、ベトナムでは『豚年』なんですよ(笑)」【2】 |
| **加藤** | 「あはは、猪じゃなくて豚!家畜の方ね、それも文化が出るね。」 |

📝【1】 12 giáp VN: **Tý/Sửu(水牛)/Dần/卯(猫)/Thìn/Tỵ/Ngọ/Mùi(山羊)/Thân/Dậu/Tuất/Hợi(豚)**. So với JP: 子・丑・寅・卯(兎)・辰・巳・午・未・申・酉・戌・亥(猪).
📝【2】 **3 con giáp VN khác JP:** Sửu = trâu (vs ushi/bò), Mão = mèo (vs u/thỏ), Hợi = lợn (vs i/lợn rừng), Mùi = dê (vs cừu) — JP có 4 điểm khác nhỏ, nhưng "mèo vs thỏ" là điểm impact nhất.

### Scenario 3 — Hoa đào Bắc / hoa mai Nam + mâm ngũ quả
*携帯で見せる*

| Speaker | Câu |
|---------|-----|
| **山本** | 「テトの飾り付けって、桜みたいなんあるん?」 |
| **ズン** | 「あります!**北は桃の花(ホアダオ)**、**南は黄色い梅(ホアマイ)**。地域で違うんです。」 |
| **山本** | 「南は梅?日本の梅と同じ?」 |
| **ズン** | 「種類は違って、**黄色い5枚花びら**で南国らしい鮮やかさです。"運と幸せ"の象徴です。」 |
| **山本** | 「ええなあ、写真見てみたい。」 |
| **ズン** | 「これが**バンザン花卉村(Làng hoa Vạn Giã)**の桃の市場で、テト前は花だけで埋まります。」 |
| **山本** | 「すごい!神戸の花市場みたい。」 |
| **ズン** | 「あと**マム・ングークア(五果盆)**って、5種類の果物を仏壇に飾る習慣があります。各果物に意味があって、例えば**カスタードアップル/釈迦頭(Cầu)・イチジク(Sung)・パパイヤ(Vừa Đủ)・ココナッツ(Dừa)・マンゴー(Xoài)**で読むと『十分に欲しいだけ』って意味になります。」 |
| **山本** | 「えっ、ダジャレ?めっちゃ関西人気質や(笑)」 |

> **VN:** Mâm ngũ quả Nam = ngôn ngữ chơi chữ "Cầu Sung Vừa Đủ Xài" → khách Kansai cười sằng sặc vì "ダジャレ" văn hóa.

### Scenario 4 — Lì xì + né chính trị
*深く印象付けられる*

| Speaker | Câu |
|---------|-----|
| **松本** | 「お正月のお年玉みたいなのある?」 |
| **ズン** | 「**リーシー(Lì xì)**っていう赤い小袋にお金を入れて、子供や年長者にも渡します。」 |
| **松本** | 「年長者にも?日本のお年玉は子供だけだよね。」 |
| **ズン** | 「そう、ベトナムは**親や祖父母にも『健康と長寿の願い』として**渡します。中の額より気持ちが大事です。」 |
| **松本** | 「いいね、温かい習慣だね。テトの家族の集まりって何日くらい?」 |
| **ズン** | 「**3が日が一番濃い**です。元日は両親、2日目は妻側の親、3日目は先生や恩人を訪ねる、というパターンが多いです。」 |
| **松本** | 「先生も訪ねるの?素敵だね。」 |
| **ズン** | 「『**師事の感謝**』を表す日で、儒教文化の名残ですね。**Mùng 3 tết Thầy(3日目は先生の日)**って言葉があるくらいです。」 |
| **松本** | 「いやあ、こういう話を聞けるのが本当に嬉しいよ。」 |

📝 **Vùng cấm:** Tết Mậu Thân 1968, dòng người từ TP HCM về quê (gợi nhớ chiến tranh & nội chiến), so sánh "Bắc-Nam ăn Tết khác". Stick với **văn hóa gia đình + ẩm thực + hoa**.

## NG — tuyệt đối tránh
- Mention "Tết Mậu Thân 1968" / chiến tranh → vùng cấm tuyệt đối.
