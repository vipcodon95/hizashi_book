# Rule 26 — Osaka — お好み焼, 阪神, 笑い文化 / 大阪
> **Luận điểm.** Osaka = "**食い倒れの街**" + "**笑いの街**" + "**阪神タイガースの街**". Khách Osaka thẳng + nhanh + thích cười + ghét formal cứng. **Dùng nhẹ Kansai-ben** ("ほんま" / "おおきに") = thân ngay, nhưng đừng overdo sẽ thành nhạo.

> **Ronten.** 大阪は食(お好み焼/たこ焼/串カツ)、阪神タイガース、関西弁、笑い文化の街。気さくでフレンドリー、フォーマル過ぎる対応はNG。

---

## Tâm lý JP
- 大阪人 nói thẳng, cười to, không câu nệ — feel "**家族みたい**" 30 phút.
- "**つっこみ・ボケ**" = 双方向 hài. Khách Osaka đợi mình react với "なんでやねん!" tone.
- 阪神 fan đặc biệt cuồng — không đụng được nếu mùa thua.
- 「**おおきに**」 = cám ơn (Kansai). 「**ほんま**」 = thật (とても/本当). 「**せやな**」 = đúng vậy.

## 4 Scenarios
### Scenario 1 — Trưa たこ焼 ở 法善寺横丁
*dẫn vào quán · 笑 · eyes wide*

| Speaker | Câu |
|---------|-----|
| **yamamoto_kansai** | 「ズンさん、リンちゃん、大阪のたこ焼はな、**外カリッと中トロッ**が命やで。**わなか**いう店、ほんま美味いから。」 |
| **ズン** | 「『ほんま』、いいですね、関西弁。」 |
| **yamamoto_kansai** | 「ほんま?嬉しいわぁ。たこ焼は東京の食べ方とちゃう、**ソース+マヨ+青のり+鰹節**、これ全部のせが大阪流。」 |
| **リン** | 「家のたこ焼器、大阪の家庭にあるって本当ですか?」 |
| **yamamoto_kansai** | 「ほんまほんま!**1家1台**って言うくらい。**たこパー (たこ焼パーティー)**って、大阪の家族の定番。」 |
| **ズン** | 「**たこパー**、初耳です(笑)」 |
| **yamamoto_kansai** | 「次回大阪来たら、家でたこパーしよ!えっと、**お好み焼き**との違いも分かる?」 |
| **ズン** | 「お好み焼は**生地+具を混ぜて焼く『混ぜ焼き』**、**広島**は**重ね焼き**ですよね?」 |
| **yamamoto_kansai** | 「正解!**広島さん**(=Hiroshi さん)から教わったんやな?やるやん。」 |

> **VN:** "ほんま" + "ちゃう" + "ええやん" + "やるやん" = Kansai natural. Phân biệt 大阪混ぜ焼き / 広島重ね焼き = depth okonomiyaki. **たこパー** = từ đặc trưng.

### Scenario 2 — Chiều training — 笑い văn hóa
*thực tập sinh, hỏi nhỏ · ずっこける · 笑*

| Speaker | Câu |
|---------|-----|
| **リン** | 「山本さん、大阪では『ボケ』と『つっこみ』、本当に日常にあるんですか?」 |

> **VN:** ボケ・つっこみ structure: "なんでやねん" / "自分で言うんかい!" — đây là module hài chuẩn. Phân biệt **東京=信頼で売る ↔ 大阪=人柄で売る** = insight commercial vùng.

## NG
- **Overdo Kansai-ben** ("わいは〜やで!") → coi nhạo người Osaka.
- Mặt **lạnh + formal cứng** với khách Osaka → bị cho là "**冷たい**" / 東京っぽい.
- Khen **「東京の方が洗練されてる」** trước khách Osaka → conflict cố hữu.
- Trêu Hanshin **khi đang thua mùa** → chạm tự ái cuồng.
- Nhầm **大阪お好み焼 ↔ 広島お好み焼** → 2 vùng kình nhau.
- Hỏi quá thân **năm đầu** khi mới làm → người Osaka thân nhanh nhưng cấp độ formal vẫn cần ban đầu.
